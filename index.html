Mongounit is a PHPUnit extension for test cases that utilize MongoDB as their data source. An example test case has been provided to show the ease of use.

Current Version: v1.1

Requirements

  • PHP 5.3+
  • PECL mongo 1.3+

Testing

After cloning the repo, simply cd to the cloned directly and run:

$ phpunit
