Mongounit is a PHPUnit extension for test cases that utilize MongoDB as their data source.

Requirements

  • PHP 5.3+
  • PECL mongo 1.3+

Testing

  1. Clone repo.
  2. Install dependencies composer install -dev
  3. Run ./bin/phpunit
